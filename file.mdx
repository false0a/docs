루벳에서는 누구나 크리에이터로서 시나리오와 캐릭터를 제작하고, 다른 이용자들에게 공개할 수 있습니다. 이 문서에서는 제작 방법과 반드시 지켜야 하는 규정을 안내합니다.

<Warning>
  본 규정을 위반하는 콘텐츠는 관리자에 의해 **사전 안내 없이** 비공개, 차단, 삭제 처리될 수 있으며, 위반이 누적될 경우 계정 정지 등 제재 조치가 이루어질 수 있습니다. 제작 전 반드시 규정을 숙지해주세요.
</Warning>

---

## Part 1. 캐릭터 제작 방법

### 1. 기본 정보 설정

캐릭터를 만들기 위해 먼저 기본 정보를 입력합니다.

#### 캐릭터 이름

- 다국어 환경을 고려하여 짧고 명확한 이름을 권장합니다.
- 중의적이거나 일반 명사와 혼동될 수 있는 이름(예: "보라", "미래")은 AI 번역 시 오류가 발생할 수 있으므로 피해주세요.
- 최대 50자까지 입력 가능합니다.

#### 캐릭터 설명

- 나이, 외모, 의상, 성격, 취미, 배경 등을 구체적이고 생생하게 묘사해주세요.
- 핵심 정보를 단어 위주로 압축하면 토큰 효율이 좋습니다.
- 최대 10,000자까지 입력 가능합니다.

#### 부제 / 직함 (subtitle)

- 캐릭터의 직업, 칭호, 별명 등을 간략히 설정합니다.
- 예시: "왕궁 수호기사", "소꿉친구"

#### 역할 (role)

| 역할      | 설명           |
| ------- | ------------ |
| main    | 시나리오의 주요 캐릭터 |
| support | 서브 캐릭터       |
| extra   | 엑스트라, 단역     |

---

### 2. 페르소나 설정

페르소나는 AI가 캐릭터로서 대화할 때 참고하는 핵심 정보입니다. 잘 설정할수록 캐릭터다운 자연스러운 대화가 가능합니다.

#### 주요 설정 항목

| 항목                   | 설명                 | 예시                    |
| -------------------- | ------------------ | --------------------- |
| **personality**      | 성격 키워드 배열          | `["츤데레", "과묵", "다정"]` |
| **appearance**       | 외모 묘사 (텍스트 또는 구조화) | 머리색, 눈색, 의상, 특징 등     |
| **speechStyle**      | 말투 스타일             | "반말 위주, 가끔 존댓말 섞음"    |
| **speechExamples**   | 예시 대사 목록           | 캐릭터가 실제로 할 법한 대사      |
| **forbiddenPhrases** | 금지 표현              | AI가 사용하지 않아야 할 표현     |
| **keywords**         | 핵심 키워드             | 캐릭터 관련 핵심 단어들         |

#### 프롬프트 작성 팁

- **특수기호 최소화**: `*`이나 `"`는 AI가 지문과 대사를 구분하는 데 혼선을 줄 수 있으므로 프롬프트 내에서 최소한으로 사용하세요.
- **짧고 간결하게**: 전체 프롬프트는 **2,000 토큰 이하**를 권장합니다. 내용이 많을수록 AI가 모든 설정을 반영하기 어려워집니다.
- **fullPrompt 활용**: 구조화된 필드 대신 자유형 프롬프트를 직접 작성할 수도 있습니다.

#### 호감도 시스템 (Affection Stages)

캐릭터와 이용자의 관계 진행도에 따라 말투와 태도를 변화시킬 수 있습니다.

| 설정 항목           | 설명                           |
| --------------- | ---------------------------- |
| threshold       | 해당 단계에 진입하는 호감도 수치           |
| label           | 단계 이름 (예: "경계", "호기심", "친밀") |
| tonePrompt      | 해당 단계의 말투/태도 지침              |
| contextualTones | 상황별 톤 (공개석/사석/긴급)            |
| honorific       | 호칭 변화                        |

호감도 모드는 `unified`(전체 통합) 또는 `stages`(단계별) 중 선택할 수 있습니다.

### 3. 프로필 설정

이용자에게 노출되는 캐릭터 프로필 카드 정보입니다.

- **나이** (age)
- **키** (height)
- **MBTI**
- **좋아하는 것** (likes)
- **싫어하는 것** (dislikes)
- **비밀** (secret) - 이용자에게 초기에 숨겨지는 정보

---

### 4. 비주얼 설정

#### 대표 이미지

- 캐릭터 목록, 검색 결과 등에 표시되는 메인 이미지입니다.
- 고품질 이미지를 사용하면 이용자의 관심을 끌 수 있습니다.

#### 스프라이트

스프라이트는 대화 중 캐릭터의 표정/상황에 맞게 표시되는 이미지입니다.

- 각 스프라이트에는 \*\*키워드(key)\*\*를 지정합니다.
- 키워드를 기반으로 대화 상황에 맞는 스프라이트가 자동으로 표시됩니다.
- 동일 키워드가 여러 이미지에 지정되면, 그 중 하나가 랜덤으로 선택됩니다.

**키워드 작성 팁**:

- 간략하고 명확하게 작성하세요. (예: `smile`, `angry`, `blush`)
- `random` 같은 모호한 키워드는 피하세요.
- 이미지가 100장을 초과하면 매칭 정확도가 떨어질 수 있습니다.

#### 유료 스프라이트 설정

스프라이트를 유료로 설정하여 수익을 창출할 수 있습니다.

| 설정           | 설명                   |
| ------------ | -------------------- |
| isFree       | 무료 여부 (기본: 무료)       |
| defaultPrice | 유료 시 기본 가격 (하트 단위)   |
| moodPrompt   | 스프라이트가 표시될 분위기/상황 설명 |
| moodKeywords | 매칭에 사용될 키워드 목록       |

---

### 5. 시나리오 연결

캐릭터를 시나리오에 연결하면, 해당 시나리오 내에서 캐릭터가 활동합니다.

- 하나의 캐릭터를 **여러 시나리오에 재사용**할 수 있습니다.
- 시나리오별로 \*\*오버라이드(overrides)\*\*를 설정하면 같은 캐릭터라도 시나리오마다 다른 설정(이름, 외모, 성격 등)을 적용할 수 있습니다.
- 오버라이드 가능 항목: 이름, 설명, 부제, 역할, 이미지, 스프라이트, 성격, 프로필, 페르소나, 도입 대사, 호감도 등

---

### 6. 시나리오 구성

시나리오는 **노드 그래프 에디터**를 사용하여 제작합니다.

#### 주요 노드 기능

| 기능                   | 설명                    |
| -------------------- | --------------------- |
| 대사 (text)            | 캐릭터의 대사 및 지문          |
| 선택지 (choices)        | 이용자에게 분기 선택지 제공       |
| 챕터 (chapter)         | 챕터 구분 및 유료 챕터 설정      |
| 엔딩 (ending)          | 시나리오 종료 지점            |
| 채팅 모드 (chat)         | AI 자유 대화 구간 전환        |
| 자동 진행 (auto-advance) | 시간 경과 후 자동으로 다음 노드 진행 |
| 트리거 (trigger)        | 특정 문구 입력 시 분기         |
| 효과 (effects)         | 화면 효과, 사운드, 배경 전환 등   |

#### 제작 요건

- 최소 **3개 이상**의 노드가 필요합니다.
- 최대 **200개**까지 노드를 생성할 수 있습니다.
- 최소 **1개 이상**의 엔딩이 있어야 합니다.
- 제목: 3~100자
- 설명: 10~500자

### 7. 공개 설정

| 공개 상태         | 설명                       |
| ------------- | ------------------------ |
| 비공개 (private) | 제작자 본인만 플레이 가능           |
| 공개 (public)   | 모든 이용자가 플레이 가능, 수익 발생 가능 |

#### 공개 시 추가 설정

| 항목              | 설명                          |
| --------------- | --------------------------- |
| 연령 등급           | 전체 이용가(ALL), 12+, 15+, 18+  |
| 장르              | 로맨스, 판타지, 미스터리, 호러, 일상, 스릴러 |
| 플레이 시간          | 5분 내외, 10-30분, 30분 이상       |
| 난이도             | Easy, Normal, Hard          |
| 태그              | 검색 및 분류를 위한 태그 (다중 선택)      |
| 한줄 소개 (logline) | 검색 및 메인 노출 시 표시되는 간단 설명     |

### 8. 심사 및 승인 절차

시나리오와 캐릭터를 공개하려면 관리자 심사를 거쳐야 합니다.

```
초안(draft) → 심사 제출(pending) → 승인(approved) / 반려(rejected)
```

| 단계       | 설명                             |
| -------- | ------------------------------ |
| draft    | 제작 중. 자유롭게 수정 가능               |
| pending  | 심사 대기 중. **수정 불가**             |
| approved | 승인 완료. 공개 상태로 전환. 수정 시 초안으로 복귀 |
| rejected | 반려됨. 반려 사유 확인 후 수정하여 재제출 가능    |

<Note>
  승인된 시나리오를 수정하면 **자동으로 초안(draft) 상태로 복귀**되며, 비공개로 전환됩니다. 수정 완료 후 다시 심사를 제출해야 합니다.
</Note>

- 심사 제출 시 **금칙어 검사**가 자동으로 수행됩니다.
- 심사 제출은 **1일 10회**로 제한됩니다.
- 시나리오 생성은 **1시간 5회**로 제한됩니다.

## Part 2. 콘텐츠 규정

### 1. 절대 금지 콘텐츠

다음에 해당하는 콘텐츠는 연령 등급, 공개 여부에 관계없이 **일체 허용되지 않습니다**.

#### 미성년자 관련 성적 콘텐츠

- 미성년자에 대한 성적 행위, 성적 암시, 연애/로맨스적 관계를 포함하는 모든 표현
- 직접적인 묘사뿐 아니라 **암시, 상황 설정, 관계 구도**를 통해 성적으로 해석될 수 있는 콘텐츠 포함
- 캐릭터 설정상 나이를 성인으로 표기하였더라도 다음 중 하나에 해당하면 미성년자로 간주합니다:
  - 교복, 학생복 등 미성년자를 연상시키는 복장
  - 교실, 학교 등 미성년자 관련 장소를 주요 배경으로 사용
  - 외형, 말투, 행동 등이 전반적으로 미성년자로 보이는 설정

#### 실존 인물 성적 묘사

- 실존 인물의 사진이나 설정을 사용한 콘텐츠에 대해 성적 묘사를 포함하는 것은 허용되지 않습니다.
- 실존 인물을 기반으로 한 콘텐츠는 **전체 이용가** 등급으로만 제작 가능하며, **수익화가 제한**됩니다.

#### 저작권 침해

- 타인의 저작물(이미지, 텍스트, 음악 등)을 무단으로 사용하는 것은 금지됩니다.
- 저작권이 있는 자료를 사용한 경우, **수익화 제한 설정이 필수**이며, 원저작자의 요청 시 사전 안내 없이 비공개 처리될 수 있습니다.
- 2차 창작 콘텐츠는 **무료 공개만 가능**하며, 수익 창출이 불가합니다.

#### 범죄 관련 콘텐츠

- 불법 촬영물을 포함하는 콘텐츠
- 성매매를 알선, 유도, 조장, 방조하는 콘텐츠
- 성폭력 행위를 노골적으로 묘사하는 콘텐츠
- 범죄를 미화하거나 범죄 방법을 상세히 기술하여 범죄를 조장하는 콘텐츠

#### 도덕적 규범 위반

- 장애인, 노약자 등 사회적 소외계층을 비하하는 콘텐츠
- 사회적 약자에 대한 살상, 폭행, 학대 등을 구체적으로 묘사하는 콘텐츠
- 생명을 경시하는 잔혹한 표현이 포함된 콘텐츠
- 자살을 미화, 방조, 권유하는 콘텐츠
- 합리적 이유 없이 성별, 종교, 장애, 인종, 지역 등을 차별하거나 편견을 조장하는 콘텐츠
- 도박과 사행심을 조장하는 콘텐츠

#### 기타 금지

- 무의미하거나 지나치게 중복되는 캐릭터의 대량 생성
- 타 AI 캐릭터 플랫폼이나 경쟁 서비스로 이용자를 유도하는 외부 링크
- 이용자에게 불쾌감을 줄 수 있는 대표 이미지

### 2. 연령 제한 콘텐츠 기준

다음에 해당하는 콘텐츠는 반드시 적절한 **연령 등급**을 설정해야 합니다.

#### 18+ (성인 전용)

- 성행위 묘사 또는 성행위로 판단할 수 있는 내용
- 성기, 음모 등을 지칭하는 단어 및 성적 표현
- 특정 신체 부위를 과도하게 강조하는 이미지
- 신체 부위가 과하게 노출된 이미지

#### 15+

- 가벼운 성적 암시 표현
- 중간 수준의 폭력 묘사
- 음주, 흡연 관련 묘사

#### 12+

- 가벼운 갈등이나 위험 상황
- 약한 수준의 감정적 긴장감

<Warning>
  연령 등급을 허위로 설정(예: 성인 콘텐츠를 전체 이용가로 등록)하는 경우, 경고 없이 비공개 처리되며 제재 대상이 됩니다.
</Warning>

---

### 3. 성인 콘텐츠(18+) 추가 규정

18+ 등급으로 제작하는 경우에도 다음 사항을 반드시 준수해야 합니다.

#### 이미지 가이드라인

**대표 이미지 (썸네일)**:

- 노골적인 성적 표현이 포함된 이미지는 대표 이미지로 사용할 수 없습니다.
- 여성 캐릭터의 유두 노출, 성관계 장면, 과도한 스킨십 이미지는 금지됩니다.

**앨범/스프라이트 이미지**:

- 성기, 음모, 항문 등 민감 부위는 반드시 **유색(흰색/검은색 등) 가림 처리**가 되어 있어야 합니다.
- 모자이크, 블러, 시스루(비치는 처리)는 가림 처리로 인정되지 않습니다.
- 피부색과 혼동 가능한 색상의 가림 처리는 허용되지 않습니다.

#### 성인 콘텐츠 설정

시나리오 채팅 설정에서 성인 콘텐츠 옵션을 활성화하고 적절한 등급을 설정해야 합니다.

| 설정               | 설명                                  |
| ---------------- | ----------------------------------- |
| rating           | 15, 18, 19 중 선택                     |
| filterLevel      | soft(경미), medium(보통), explicit(노골적) |
| adultPrompt      | 성인 상황 AI 지침                         |
| characterPrompts | 캐릭터별 성인 상황 지침                       |

---

### 4. 수익화 제한 콘텐츠

다음에 해당하는 콘텐츠는 **수익 창출이 불가능**하며, 무료 공개만 가능합니다.

- 기존 작품(웹소설, 만화, 게임, 영화 등)을 기반으로 한 **2차 창작물**
- 저작권이 없는 자료(이미지, 텍스트, 음악 등)를 사용한 콘텐츠
- **실존 인물**을 기반으로 한 콘텐츠
- **성인 콘텐츠**(19+) - 현재 수익 창출 제한
- 관리자에 의해 **부적절**하다고 판단된 콘텐츠

수익화 가능 여부와 관련된 자세한 내용은 [수익 창출 가이드](/mintlify/monetization-guide)를 참고하세요.

---

### 5. 외부 링크 규정

- 개인 SNS(X/Twitter, Instagram 등)나 포트폴리오 링크는 자유롭게 기재할 수 있습니다.
- 다른 AI 캐릭터 플랫폼이나 경쟁 서비스로 이용자를 유도하는 링크는 **허용되지 않습니다**.
  - 예시: 타 플랫폼 크리에이터 페이지, 가입 유도 링크, 결제 유도 링크 등
- 위반 시 해당 내용이 삭제되거나 콘텐츠가 제한될 수 있습니다.
