## 1. 수익 구조 한눈에 보기

루벳의 수익 구조는 다음과 같이 작동합니다.

```
사용자가 하트를 소비 → 크리에이터에게 피스(수익)가 적립 → 피스를 하트로 전환하여 사용
```

크리에이터가 만든 **캐릭터**와 **시나리오**를 다른 사용자가 이용할 때, 소비된 하트의 일정 비율이 크리에이터에게 **피스**로 돌아옵니다.

---

## 2. 통화 시스템 이해하기

루벳에는 두 가지 통화가 있습니다.

### 하트

사용자가 콘텐츠를 이용할 때 소비하는 기본 통화입니다.

| 구분    | 설명                                         |
| ----- | ------------------------------------------ |
| 무료 하트 | 가입 보너스, 이벤트, 피스 전환 등으로 획득합니다. 환불 대상이 아닙니다. |
| 유료 하트 | 스토어에서 실제 결제하여 구매합니다. 조건 충족 시 환불이 가능합니다.    |

하트가 소비될 때 **무료 하트가 먼저** 차감되고, 부족한 만큼 유료 하트가 차감됩니다.

**하트 사용처:**

| 사용처       | 비용                       |
| --------- | ------------------------ |
| AI 채팅 메시지 | AI 모델에 따라 메시지당 N 하트      |
| 유료 챕터 해금  | 시나리오 설정에 따라 다름           |
| 스프라이트 구매  | 일반 100하트, 영상 500하트 (기본값) |

### 피스

크리에이터의 수익 통화입니다.

- 사용자가 내 콘텐츠에 하트를 소비하면, 일정 비율이 피스로 적립됩니다.
- 적립된 피스는 하트로 전환하여 사용할 수 있습니다.

---

## 3. 수익을 얻는 3가지 방법

### 3-1. 챕터 판매 수익

시나리오를 "부분 유료"로 설정하면, 사용자가 유료 챕터를 해금할 때 수익이 발생합니다.

| 항목          | 내용                               |
| ----------- | -------------------------------- |
| 크리에이터 배분 비율 | 판매가의 **30%**                     |
| 플랫폼 수수료     | 70%                              |
| 무료 제공 챕터    | 첫 번째 챕터는 기본 무료 (설정 변경 가능)        |
| 가격 설정       | 시나리오 전체 기본 가격 또는 챕터별 개별 가격 설정 가능 |

**예시:** 사용자가 300하트짜리 챕터를 해금하면, 크리에이터에게 90피스가 적립됩니다.

### 3-2. 스프라이트 판매 수익

캐릭터에 유료 스프라이트(일러스트)를 설정해 두면, 사용자가 해당 스프라이트를 구매할 때 수익이 발생합니다.

| 항목          | 내용                        |
| ----------- | ------------------------- |
| 크리에이터 배분 비율 | 판매가의 **50%**              |
| 플랫폼 수수료     | 50%                       |
| 기본 가격       | 일반 이미지: 100하트 / 영상: 500하트 |
| 가격 조정       | 크리에이터가 스프라이트별로 자유롭게 설정 가능 |

**스프라이트 해금 방식:**

- 채팅 중 AI가 상황에 맞는 스프라이트를 자동으로 표시합니다.
- 키워드가 매칭되면 해당 스프라이트가 **영구 해금**됩니다.
- 영구 해금된 스프라이트는 일러스트 앨범에서도 볼 수 있습니다.

**팁:** 특별 의상, 레어 표정, 한정판 일러스트 등 희소성 있는 스프라이트에 높은 가격을 설정하면 효과적입니다.

### 3-3. 채팅 수익

사용자가 내 시나리오의 캐릭터와 AI 채팅을 할 때, 메시지당 소비되는 하트의 일부가 수익으로 적립됩니다.

| 항목          | 내용                      |
| ----------- | ----------------------- |
| 크리에이터 배분 비율 | 소비 하트의 **10%**          |
| 플랫폼 수수료     | 90%                     |
| 적립 시점       | 사용자가 메시지를 보낼 때마다 실시간 적립 |

**예시:** 사용자가 30하트짜리 메시지를 보내면, 크리에이터에게 3피스가 적립됩니다.

> **참고:** 본인이 직접 자신의 시나리오에서 채팅하는 경우에는 수익이 발생하지 않습니다. 다른 사용자가 이용할 때만 적용됩니다.

### 수익 배분 요약

| 수익원      | 크리에이터 | 플랫폼 |
| -------- | ----- | --- |
| 챕터 판매    | 30%   | 70% |
| 스프라이트 판매 | 50%   | 50% |
| 채팅 메시지   | 10%   | 90% |

## 4. 피스를 하트로 전환하기

적립된 피스는 하트로 전환할 수 있습니다.

| 항목        | 내용                  |
| --------- | ------------------- |
| 전환 비율     | **1피스 = 1하트** (1:1) |
| 최소 전환 금액  | **100피스**           |
| 전환된 하트 종류 | 무료 하트로 지급           |

### 전환 방법

1. **제작 스튜디오** \> **수익** 페이지로 이동합니다.
2. "피스 전환" 영역에서 전환할 피스 수량을 입력합니다.
3. "전환하기" 버튼을 클릭합니다.
4. 즉시 무료 하트로 전환되어 잔액에 반영됩니다.

## 5. 수익 대시보드 사용법

수익 현황은 **제작 스튜디오 \> 수익** 페이지에서 확인할 수 있습니다.

### 개요 탭

세 가지 핵심 지표가 상단에 표시됩니다.

| 지표      | 설명              |
| ------- | --------------- |
| 보유 피스   | 현재 전환 가능한 피스 잔액 |
| 이번 달 수익 | 이번 달에 적립된 총 피스  |
| 누적 수익   | 가입 이후 적립된 총 피스  |

**수익원별 상세** 섹션에서는 각 수익원의 실적을 확인할 수 있습니다.

- **시나리오 챕터 판매** - 총 수익, 판매 건수
- **캐릭터 대화 수익** - 총 수익, 메시지 수
- **스프라이트 판매** - 총 수익, 판매 건수

**인기 시나리오** 섹션에서는 수익이 높은 시나리오 순위를 확인할 수 있습니다.

### 내역 탭

모든 수익 및 전환 기록을 시간순으로 확인할 수 있습니다.

| 유형       | 표시      |
| -------- | ------- |
| 챕터 판매    | 초록색 (+) |
| 채팅 수익    | 초록색 (+) |
| 스프라이트 판매 | 초록색 (+) |
| 하트 전환    | 빨간색 (-) |

유형별 필터링도 지원되어, 특정 수익원의 내역만 따로 볼 수 있습니다.

## 6. 수익 창출 조건

수익을 얻으려면 다음 조건을 모두 충족해야 합니다.

1. **콘텐츠가 공개 상태여야 합니다.**
   - 비공개(나만 보기) 상태인 시나리오/캐릭터에서는 수익이 발생하지 않습니다.
   - 관리자 심사를 통과하여 "승인됨" 상태여야 합니다.
2. **크리에이터 선언 체크리스트를 완료해야 합니다.**
   - 유료 콘텐츠를 제공하려면 5가지 항목 모두에 동의해야 합니다.
   - 체크리스트를 완료하지 않으면 무료 공개만 가능합니다.
3. **다른 사용자가 이용해야 합니다.**
   - 본인이 직접 이용하는 경우에는 수익이 발생하지 않습니다.

---

## 7. 크리에이터 선언 체크리스트

유료 콘텐츠를 등록할 때 다음 5가지 항목에 모두 동의해야 합니다.

| 항목                | 설명                                                 |
| ----------------- | -------------------------------------------------- |
| 본인의 창작물입니다        | 이 콘텐츠는 직접 만든 오리지널 작품입니다.                           |
| 모든 저작권을 보유하고 있습니다 | 사용된 모든 이미지, 텍스트, 음악 등에 대한 저작권을 보유하거나 사용 권한이 있습니다.  |
| 2차 창작이 아닙니다       | 기존 작품(웹소설, 만화, 게임, 영화 등)을 기반으로 한 팬픽이나 2차 창작이 아닙니다. |
| 성인 콘텐츠가 아닙니다      | 19세 미만이 이용할 수 없는 성적, 폭력적 콘텐츠가 포함되지 않았습니다.          |
| 커뮤니티 가이드라인을 준수합니다 | 플랫폼의 커뮤니티 가이드라인과 이용약관을 준수합니다.                      |

> 하나라도 체크하지 않으면 유료 판매가 불가능하며, **무료로만 공개**할 수 있습니다.

## 8. 수익 창출이 제한되는 경우

다음에 해당하는 콘텐츠는 수익 창출이 제한됩니다.

| 사유       | 설명                                          |
| -------- | ------------------------------------------- |
| 2차 창작    | 기존 작품을 기반으로 한 팬픽, 패러디 등은 무료 공개만 가능합니다.      |
| 성인 콘텐츠   | 19세 이상 전용 성적/폭력적 콘텐츠는 수익 창출이 제한됩니다.         |
| 저작권 침해   | 타인의 이미지, 텍스트, 음악 등을 무단으로 사용한 경우 제재될 수 있습니다. |
| 가이드라인 위반 | 커뮤니티 가이드라인을 위반한 콘텐츠는 삭제되거나 수익이 차단될 수 있습니다.  |
| 관리자 판단   | 위 항목 외에도 관리자 판단에 의해 수익 창출이 제한될 수 있습니다.      |

수익 창출이 차단되면 해당 콘텐츠는 자동으로 **무료 전환**됩니다.

## 9. 자주 묻는 질문

### Q. 피스를 현금으로 출금할 수 있나요?

현재는 피스를 하트로 전환하여 플랫폼 내에서 사용하는 방식만 지원됩니다. 현금 출금 기능은 추후 업데이트 예정입니다.

### Q. 수익은 실시간으로 반영되나요?

채팅 수익은 메시지가 발송될 때마다 실시간으로 적립됩니다. 챕터 판매와 스프라이트 판매도 구매 즉시 반영됩니다.

### Q. 비공개 상태에서도 수익이 발생하나요?

아닙니다. 시나리오가 공개(승인됨) 상태이고, 본인이 아닌 다른 사용자가 이용할 때만 수익이 발생합니다.

### Q. 수익 배분 비율은 변경될 수 있나요?

수익 배분 비율은 플랫폼 정책에 따라 변경될 수 있습니다. 변경 시 사전에 공지됩니다.

### Q. 하트 최대 보유량이 있나요?

하트 최대 보유량은 1,000,000 하트입니다.

### Q. 크리에이터 선언을 나중에 변경할 수 있나요?

캐릭터/시나리오를 수정할 때 체크리스트를 다시 작성할 수 있습니다.
