---

루벳에서는 캐릭터와 시나리오가 **독립적으로 관리**됩니다. 하나의 캐릭터를 여러 시나리오에서 활용할 수 있고, 시나리오마다 캐릭터의 설정을 다르게 조정할 수 있습니다. 이 문서에서는 캐릭터를 시나리오에 추가하고 관리하는 모든 방법을 안내합니다.

---

## 구조 이해하기

루벳의 캐릭터 시스템은 크게 두 부분으로 나뉩니다.

### 캐릭터 라이브러리

**스튜디오 \> 캐릭터 관리**에서 만드는 캐릭터입니다.

- 시나리오와 무관하게 **독립적으로 존재**합니다.
- 이름, 외모, 성격, 스프라이트 등 **원본 설정**을 여기서 관리합니다.
- 여러 시나리오에서 재사용할 수 있습니다.

### 시나리오 내 캐릭터

특정 시나리오에 캐릭터를 추가하면, **원본 캐릭터와의 연결 정보**가 생깁니다.

- 이 연결에는 시나리오 안에서만 적용되는 **개별 설정**을 저장할 수 있습니다.
- 같은 캐릭터라도 시나리오마다 다른 이름, 외모, 성격 등을 적용할 수 있습니다.
- 원본 캐릭터를 수정하면, 개별 설정으로 덮어쓰지 않은 항목은 자동으로 반영됩니다.

```
캐릭터 라이브러리          시나리오 A             시나리오 B
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  "시온"       │────▶│  시온 (원본)   │     │  시온 (카페)   │
│  원본 설정     │     │  + 개별 설정 X │────▶│  + 개별 설정 Y │
└──────────────┘     └──────────────┘     └──────────────┘
```

## 시나리오에 캐릭터 추가하기

캐릭터를 시나리오에 추가하는 방법은 두 가지입니다.

### 방법 1: 라이브러리에서 기존 캐릭터 추가

이미 만들어둔 캐릭터를 시나리오에 불러옵니다.

1. 시나리오 에디터에서 **캐릭터 관리** 패널을 엽니다.
2. **캐릭터 추가** 버튼을 클릭합니다.
3. 내 캐릭터 라이브러리에서 추가할 캐릭터를 선택합니다.
4. 필요하면 이 시나리오에서만 적용할 개별 설정을 바로 지정할 수 있습니다.

<Note>
  본인이 만든 캐릭터만 추가할 수 있습니다. 같은 캐릭터를 하나의 시나리오에 중복으로 추가할 수는 없습니다.
</Note>

### 방법 2: 시나리오 안에서 새 캐릭터 만들기

시나리오 편집 중에 즉석에서 새 캐릭터를 만들 수 있습니다.

1. 시나리오 에디터에서 **캐릭터 추가 \> 새로 만들기**를 선택합니다.
2. 캐릭터 이름과 기본 정보를 입력합니다.
3. 새 캐릭터가 **라이브러리에 자동 등록**됨과 동시에 시나리오에 추가됩니다.

이렇게 만든 캐릭터는 라이브러리에도 저장되므로, 나중에 다른 시나리오에서도 사용할 수 있습니다. 처음에는 **비공개 + 작성 중** 상태로 생성됩니다.

## 개별 설정

시나리오마다 캐릭터의 설정을 다르게 적용하고 싶을 때 **개별 설정**을 사용합니다. 원본 캐릭터는 그대로 유지하면서, 이 시나리오에서만 특정 항목을 바꿀 수 있습니다.

### 변경할 수 있는 항목

| 항목       | 설명                     |
| -------- | ---------------------- |
| 이름       | 시나리오 안에서 사용할 다른 이름     |
| 설명       | 시나리오에 맞는 캐릭터 설명        |
| 부제 / 직함  | 이 시나리오에서의 역할이나 칭호      |
| 역할       | 주연, 조연, 엑스트라           |
| 대표 이미지   | 이 시나리오에서 사용할 다른 이미지    |
| 스프라이트    | 대화 중 표시되는 캐릭터 이미지 세트   |
| 스프라이트 설정 | 스프라이트 매칭 설정            |
| 성격 키워드   | 이 시나리오에서의 성격           |
| 프로필      | 나이, 키, 좋아하는 것 등 프로필 정보 |
| 페르소나     | AI 대화에 사용되는 캐릭터 설정     |
| 도입 대사    | 이 시나리오에서 캐릭터가 처음 하는 말  |
| 호감도 단계   | 호감도에 따른 태도 변화 설정       |

### 설정이 적용되는 방식

개별 설정의 항목 유형에 따라 적용 방식이 다릅니다.

#### 단순 항목 (이름, 설명, 부제, 역할, 이미지, 도입 대사 등)

원본 값을 **완전히 대체**합니다.

- 예: 원본 이름이 "시온"인데 개별 설정에서 "사이온"으로 바꾸면, 이 시나리오에서는 "사이온"으로만 표시됩니다.

#### 목록 항목 (성격, 스프라이트 설정, 호감도 단계)

목록 전체를 **통째로 교체**합니다. 원본 목록에 추가하는 것이 아니라, 개별 설정의 목록으로 완전히 바뀝니다.

- 예: 원본 성격이 ["츤데레", "과묵"]인데 개별 설정에서 ["다정", "수줍음"]으로 바꾸면, 이 시나리오에서는 "다정, 수줍음"만 적용됩니다.

#### 복합 항목 (페르소나, 스프라이트, 프로필, 이미지 설정)

원본의 기존 설정을 유지하면서, **변경한 부분만 덮어씁니다**.

- 예: 원본 페르소나에 "말투: 반말 위주"와 "성격: 츤데레"가 설정되어 있는데, 개별 설정에서 "말투: 존댓말"만 바꾸면 → 이 시나리오에서는 "말투: 존댓말" + "성격: 츤데레"가 됩니다.

<Info>
  개별 설정을 비우면(초기화하면) 원본 캐릭터의 설정이 그대로 사용됩니다.
</Info>

## 대사에 캐릭터 지정하기

시나리오 에디터에서 대사를 작성할 때, 각 대사 줄마다 **어떤 캐릭터가 말하는지** 지정합니다.

### 지정 방법

1. 속성 패널의 **대사 목록** 섹션에서 대사를 선택합니다.
2. 각 대사 줄 왼쪽의 **캐릭터 드롭다운**을 클릭합니다.
3. 시나리오에 추가된 캐릭터 목록에서 선택합니다.
4. 지문이나 나레이션이면 **"지문/나레이션"** 을 선택합니다.

캐릭터를 지정하면:

- 대사 앞에 캐릭터 이름이 표시됩니다.
- 해당 캐릭터의 스프라이트(표정 이미지)가 자동으로 표시됩니다.
- 에디터에서 캐릭터별로 색상이 구분되어 가독성이 높아집니다.

<Warning>
  시나리오에 추가하지 않은 캐릭터를 대사에 지정하면 경고가 표시됩니다. 대사에 사용할 캐릭터는 먼저 시나리오에 추가해주세요.
</Warning>

## 표시 순서 관리

시나리오에 추가된 캐릭터의 **표시 순서**를 관리할 수 있습니다.

- 표시 순서는 캐릭터 관리 패널의 목록 순서로 결정됩니다.
- 드래그 앤 드롭으로 순서를 변경할 수 있습니다.
- 추가한 순서대로 자동 번호가 매겨지며, 나중에 자유롭게 변경 가능합니다.
- 이 순서는 캐릭터 선택 드롭다운, 캐릭터 목록 표시 등에 반영됩니다.

---

## 캐릭터 재사용

하나의 캐릭터를 여러 시나리오에서 활용할 수 있습니다.

### 재사용 시 장점

- **일관된 캐릭터 관리**: 원본 캐릭터의 기본 설정을 한 곳에서 관리합니다.
- **시나리오별 맞춤 설정**: 개별 설정으로 시나리오마다 다른 버전의 캐릭터를 만들 수 있습니다.
- **자동 반영**: 원본 캐릭터를 수정하면, 개별 설정으로 덮어쓰지 않은 항목은 모든 시나리오에 자동 반영됩니다.

### 활용 예시

| 시나리오   | 같은 캐릭터의 다른 모습         |
| ------ | --------------------- |
| 학원 로맨스 | "시온" - 같은 반 친구, 교복 차림 |
| 판타지 모험 | "시온" - 마법사 동료, 로브 차림  |
| 카페 일상  | "시온" - 단골 손님, 캐주얼 차림  |

각 시나리오에서 개별 설정으로 외모, 의상, 말투 등을 다르게 지정하면 됩니다.

## 템플릿에서 시작할 때

로맨스 또는 미스터리 템플릿으로 시나리오를 만들면, **기본 캐릭터가 자동으로 포함**됩니다.

| 템플릿  | 포함 캐릭터     |
| ---- | ---------- |
| 로맨스  | "상대방" (1명) |
| 미스터리 | "집사" (1명)  |

### 자동 처리되는 사항

1. 템플릿의 캐릭터가 **내 캐릭터 라이브러리에 새로 등록**됩니다.
2. 등록된 캐릭터가 **시나리오에 자동으로 연결**됩니다.
3. 템플릿에 미리 작성된 대사는 이 캐릭터에 자동 매핑됩니다.

<Note>
  템플릿으로 만든 캐릭터는 시작점일 뿐입니다. 이름, 성격, 외모 등을 자유롭게 수정하고, 새 캐릭터를 추가하세요.
</Note>

## 시나리오 저장 시 캐릭터 동기화

시나리오를 저장하면 캐릭터 정보가 자동으로 동기화됩니다.

### 동기화 과정

1. **기존 캐릭터 매칭**: 시나리오에 이미 연결된 캐릭터와 현재 편집 중인 캐릭터를 대조합니다.
2. **변경 사항 반영**: 역할, 표시 순서, 개별 설정 등의 변경이 저장됩니다.
3. **새 캐릭터 추가**: 편집 중에 새로 만든 캐릭터가 있으면 라이브러리에 등록하고 시나리오에 연결합니다.
4. **삭제된 캐릭터 정리**: 시나리오에서 제거한 캐릭터의 연결 정보가 삭제됩니다. (원본 캐릭터는 라이브러리에 그대로 남아 있습니다.)

이 모든 과정은 한꺼번에 처리되므로, 저장 중 오류가 나면 변경 사항이 모두 취소됩니다.

---

## 승인된 시나리오 수정 시 주의사항

<Warning>
  승인된 시나리오의 캐릭터 설정을 수정하면, 시나리오가 **자동으로 비공개 + 작성 중 상태로 복귀**합니다. 수정을 마친 후 다시 심사를 제출해야 합니다.
</Warning>

캐릭터 관련 변경 사항은 모두 시나리오 수정으로 간주됩니다:

- 캐릭터 추가 또는 삭제
- 개별 설정 변경
- 역할 또는 표시 순서 변경
- 대사의 캐릭터 지정 변경

---

## 캐릭터 연결 삭제하기

시나리오에서 캐릭터를 제거하면:

- 해당 시나리오에서의 **연결 정보와 개별 설정이 삭제**됩니다.
- **원본 캐릭터는 라이브러리에 그대로 유지**됩니다.
- 이 캐릭터가 지정된 대사가 있으면, 해당 대사의 캐릭터 지정이 해제됩니다.

---

## 자주 묻는 질문

### 원본 캐릭터를 수정하면 시나리오에도 반영되나요?

네, **개별 설정으로 덮어쓰지 않은 항목**은 자동으로 반영됩니다. 개별 설정으로 변경한 항목은 원본 수정의 영향을 받지 않습니다.

### 다른 사람의 캐릭터를 내 시나리오에 추가할 수 있나요?

현재는 **본인이 만든 캐릭터만** 시나리오에 추가할 수 있습니다.

### 캐릭터를 라이브러리에서 삭제하면 시나리오에는 어떻게 되나요?

캐릭터를 삭제하면 해당 캐릭터가 연결된 모든 시나리오에서 연결이 끊어집니다. 시나리오에서 해당 캐릭터의 대사는 캐릭터 미지정 상태가 됩니다.

### 하나의 시나리오에 몇 명까지 추가할 수 있나요?

기술적인 제한은 없지만, 너무 많은 캐릭터를 넣으면 이용자가 혼란스러울 수 있습니다. 스토리에 필요한 만큼만 추가하는 것을 권장합니다.

### 캐릭터 없이 시나리오를 만들 수 있나요?

네, 나레이션과 시스템 메시지만으로 구성된 시나리오도 가능합니다. 다만 AI 채팅 구간을 넣으려면 대화할 캐릭터가 필요합니다.
