## 1. 세계관이란?

세계관은 캐릭터와 멀티챗 콘텐츠가 공유하는 **배경 설정**입니다. 세계의 역사, 규칙, 종족, 마법 체계 등을 한 곳에 작성해 두면, 연결된 모든 캐릭터의 AI 대화에 해당 설정이 자동으로 반영됩니다.

### 세계관을 사용하면 좋은 경우

- 동일한 세계를 배경으로 여러 캐릭터를 운영할 때
- 멀티챗 콘텐츠에 일관된 세계 설정을 적용하고 싶을 때
- 캐릭터마다 세계 배경을 반복 작성하는 것이 번거로울 때

### 세계관의 구조

| 항목      | 설명                        |
| ------- | ------------------------- |
| 이름      | 세계관의 이름 (예: "아르카디아 왕국")   |
| 설명      | 세계관의 핵심 설정 (배경, 규칙, 역사 등) |
| 연결된 캐릭터 | 이 세계관에 소속된 캐릭터 목록         |

세계관 설명은 AI에게 시스템 프롬프트로 주입되어, 연결된 캐릭터가 세계관에 맞는 대화를 할 수 있도록 합니다.

## 2. 세계관 만들기

### 접근 경로

**제작 스튜디오** \> **내 세계관** 섹션 \> **새 세계관** 버튼을 클릭합니다.

### 입력 항목

| 항목     | 필수 여부 | 제한                        | 설명                      |
| ------ | ----- | ------------------------- | ----------------------- |
| 세계관 이름 | 필수    | 최대 100자                   | 세계관의 이름                 |
| 세계관 설명 | 필수    | 최대 10,000자 / **1,500 토큰** | 배경, 규칙, 역사, 종족, 마법 체계 등 |

> **참고:** 한글은 글자당 약 1.5토큰, 영문은 글자당 약 0.25토큰으로 계산됩니다. 입력란 하단에서 현재 토큰 사용량을 실시간으로 확인할 수 있습니다. 토큰 한도를 초과하면 생성 버튼이 비활성화됩니다.

### 작성 팁

세계관 설명에는 다음 내용을 포함하는 것을 권장합니다.

- **세계의 기본 구조** - 어떤 종류의 세계인지 (판타지, SF, 현대, 역사 등)
- **핵심 규칙** - 마법, 기술, 사회 체계 등 세계를 지배하는 규칙
- **주요 세력/집단** - 국가, 조직, 종족 등
- **시대/분위기** - 시간적 배경과 전반적인 분위기

**작성 예시:**

```text
# 세계 배경
중세풍 판타지 세계. 마법과 검이 공존하며, 4대 왕국이 대륙을 나누어 통치하고 있다.

# 마법 체계
- 마법은 마나 결정에서 에너지를 끌어와 사용한다
- 마법사는 기사단과 동등한 지위를 가진다
- 금지 마법(사령술, 시간 마법)을 사용하면 처벌받는다

# 주요 세력
- 아르카디아 왕국: 대륙 중앙, 마법 학술의 중심지
- 노르텐 제국: 대륙 북부, 군사 강국
- 엘프 연합: 동부 숲, 고대 마법 보유
```

## 3. 세계관 편집하기

### 접근 경로

**제작 스튜디오** \> **내 세계관** 섹션에서 편집할 세계관의 **수정** 버튼을 클릭합니다.

### 편집 가능한 항목

| 항목      | 설명                           |
| ------- | ---------------------------- |
| 세계관 이름  | 이름을 변경할 수 있습니다               |
| 세계관 설명  | 설명을 수정할 수 있습니다 (1,500 토큰 한도) |
| 연결된 캐릭터 | 캐릭터를 추가하거나 연결 해제할 수 있습니다     |

## 4. 캐릭터 연결하기

세계관에 기존 캐릭터를 연결하면, 해당 캐릭터의 AI 대화에 세계관 설정이 자동으로 반영됩니다.

### 연결 방법

1. 세계관 편집 페이지에서 **캐릭터 연결** 버튼을 클릭합니다.
2. 연결 가능한 캐릭터 목록에서 캐릭터를 선택합니다.
3. (선택) **이 세계관에서의 역할**을 입력합니다 (예: "근위기사", "궁정마법사").
4. **연결** 버튼을 클릭합니다.

### 연결 설정

| 항목  | 필수 여부 | 제한      | 설명                   |
| --- | ----- | ------- | -------------------- |
| 캐릭터 | 필수    | -       | 연결할 캐릭터를 선택합니다       |
| 역할  | 선택    | 최대 200자 | 이 세계관 안에서 캐릭터가 맡는 역할 |

### 연결 해제

연결된 캐릭터 목록에서 X 버튼을 클릭하면 연결이 해제됩니다. 연결을 해제해도 캐릭터 자체는 삭제되지 않습니다.

### 참고 사항

- 이미 연결된 캐릭터는 캐릭터 선택 목록에 표시되지 않습니다.
- 하나의 캐릭터를 여러 세계관에 연결할 수 있습니다.
- 본인이 만든 캐릭터만 연결할 수 있습니다.

## 5. 멀티챗과 세계관

멀티챗 콘텐츠를 제작할 때 세계관을 연결하면, 세계관 설명이 멀티챗의 AI 프롬프트에 함께 주입됩니다.

### 프롬프트 구조

멀티챗에서 세계관이 연결된 경우, AI에게 전달되는 프롬프트는 다음 순서로 구성됩니다.

| 순서 | 항목       | 설명                                    |
| --- | -------- | ------------------------------------- |
| 1  | 세계관 설명   | 세계의 배경, 규칙, 역사 등 (최대 1,500 토큰)        |
| 2  | 콘텐츠 프롬프트 | 멀티챗의 상황 설정, 유저 역할 등 (최대 1,500 토큰)     |
| 3  | 캐릭터 페르소나 | 각 참여 캐릭터의 개별 성격/말투 (캐릭터당 최대 1,000 토큰) |

이렇게 세계관과 콘텐츠 프롬프트, 개별 페르소나가 계층적으로 결합되어 일관된 대화가 가능해집니다.

---

## 6. 세계관 삭제하기

**제작 스튜디오** \> **내 세계관** 섹션에서 삭제할 세계관에 마우스를 올리면 나타나는 삭제 버튼을 클릭합니다.

### 삭제 시 영향

| 항목      | 영향                                    |
| ------- | ------------------------------------- |
| 연결된 캐릭터 | 세계관과의 연결만 해제됩니다. 캐릭터 자체는 삭제되지 않습니다.   |
| 멀티챗 콘텐츠 | 세계관 참조가 해제됩니다. 멀티챗 콘텐츠 자체는 삭제되지 않습니다. |

---

## 7. 자주 묻는 질문

### Q. 세계관은 필수인가요?

아닙니다. 세계관 없이도 캐릭터와 멀티챗 콘텐츠를 만들 수 있습니다. 동일한 배경을 공유하는 캐릭터가 여러 명일 때 편리한 기능입니다.

### Q. 세계관 설명의 토큰 한도가 1,500인 이유는?

세계관 설명은 캐릭터 페르소나, 콘텐츠 프롬프트 등 다른 프롬프트와 함께 AI에게 전달됩니다. 전체 프롬프트 크기를 적정 수준으로 유지하기 위해 세계관 설명에는 1,500 토큰의 한도가 설정되어 있습니다.

### Q. 하나의 캐릭터를 여러 세계관에 연결할 수 있나요?

가능합니다. 하나의 캐릭터를 여러 세계관에 연결할 수 있으며, 각 세계관에서 다른 역할을 부여할 수도 있습니다.

### Q. 세계관을 삭제하면 캐릭터도 삭제되나요?

아닙니다. 세계관을 삭제해도 연결된 캐릭터는 삭제되지 않습니다. 세계관과의 연결만 해제됩니다.

### Q. 세계관을 다른 사용자에게 공개할 수 있나요?

현재 세계관은 비공개로 본인만 사용할 수 있습니다.
