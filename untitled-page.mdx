# 수익 창출 가이드

ChapTalk에서는 크리에이터가 만든 시나리오와 캐릭터를 통해 수익을 창출할 수 있습니다. 이 가이드에서는 수익 구조, 수익 발생 조건, 정산 방법을 안내합니다.

---

## 화폐 시스템 이해하기

ChapTalk에는 두 가지 화폐가 존재합니다.

### 하트 - 이용자가 사용하는 화폐

이용자가 콘텐츠를 즐기기 위해 사용하는 화폐입니다.

- **무료 하트**: 가입 시 축하 보너스로 600개가 지급됩니다. 소비 시 무료 하트가 먼저 사용됩니다.
- **유료 하트**: 실제 결제를 통해 구매한 하트입니다.
- 시나리오 챕터 열람, 캐릭터 이미지 구매, AI 대화 등에 사용됩니다.

| 패키지 | 가격 | 받는 하트 | 추가 보너스 |
|--------|------|-----------|-------------|
| 기본 | 3,300원 | 2,500개 | - |
| 스탠다드 | 5,500원 | 4,500개 | - |
| 프리미엄 | 10,900원 | 10,000개 | - |
| 프로 | 32,000원 | 30,000개 | +3,000개 |
| 엘리트 | 54,000원 | 50,000개 | +7,500개 |
| 마스터 | 99,000원 | 90,000개 | +30,000개 |

### 피스 - 크리에이터가 받는 수익 화폐

크리에이터가 콘텐츠 판매를 통해 모으는 수익 전용 화폐입니다.

- 모은 피스는 **하트로 바꿔서** 직접 사용하거나, 추후 **현금 정산** 기능이 열리면 출금할 수 있습니다.
- 교환 비율: **1 피스 = 1 하트** (한 번에 최소 100 피스부터 교환 가능)
- 교환된 하트는 무료 하트로 지급됩니다.

---

## 어떻게 수익이 발생하나요?

크리에이터는 다음 세 가지 방법으로 피스를 모을 수 있습니다.

### 1. 유료 챕터 판매

이용자가 내 시나리오의 **유료 챕터를 열람**하면 수익이 발생합니다.

- 이용자가 지불한 하트의 **10%** 가 피스로 지급됩니다.
- 예시: 이용자가 50하트를 내고 챕터를 열면 → 크리에이터에게 **5피스** 지급

**어떻게 설정하나요?**
1. 시나리오 에디터에서 **수익화** 탭을 선택합니다.
2. 수익화 유형을 **유료**로 설정합니다.
3. 무료로 제공할 챕터 수와 챕터당 가격을 정합니다.
4. 원한다면 챕터마다 다른 가격을 지정할 수도 있습니다.

### 2. 캐릭터 이미지(스프라이트) 판매

이용자가 내 캐릭터의 **잠금된 이미지를 구매**하면 수익이 발생합니다.

- 이용자가 지불한 하트의 **10%** 가 피스로 지급됩니다.
- 예시: 이용자가 100하트를 내고 이미지를 해금하면 → 크리에이터에게 **10피스** 지급

**어떻게 설정하나요?**
1. 캐릭터 편집 화면에서 이미지를 추가합니다.
2. 잠금 아이콘을 켜서 유료로 설정합니다.
3. 원하는 가격을 입력합니다.

### 3. AI 대화 수익

이용자가 내 캐릭터와 **AI 대화를 나눌 때** 메시지마다 수익이 발생합니다.

- 이용자가 메시지당 소비한 하트의 **10%** 가 피스로 지급됩니다.
- 메시지당 하트 소비량은 AI 모델에 따라 다릅니다.

<Note>
대화 수익은 **다른 이용자**가 내 캐릭터와 대화할 때만 발생합니다. 본인이 직접 자신의 캐릭터와 대화하는 경우에는 수익이 생기지 않습니다.
</Note>

## 수익이 발생하려면?

피스가 정상적으로 쌓이려면 다음 조건을 **모두** 충족해야 합니다.

### 기본 조건

1. **시나리오가 공개 상태**여야 합니다. 비공개 시나리오에서는 수익이 발생하지 않습니다.
2. **수익화가 차단되지 않은 상태**여야 합니다.
3. **구매자와 제작자가 다른 사람**이어야 합니다. 본인이 자기 콘텐츠를 구매해도 수익은 지급되지 않습니다.

### 수익화를 켜려면 - 제작자 확인 체크리스트

수익화를 활성화하려면 아래 **5가지 항목에 모두 동의**해야 합니다.

| 확인 항목 | 내용 |
|-----------|------|
| 본인 창작물 | 이 콘텐츠는 제가 직접 만든 오리지널 작품입니다. |
| 저작권 보유 | 사용된 모든 이미지, 텍스트 등에 대한 권리를 갖고 있습니다. |
| 2차 창작 아님 | 기존 작품을 기반으로 한 팬픽이나 2차 창작이 아닙니다. |
| 성인 콘텐츠 아님 | 19세 미만이 볼 수 없는 성적/폭력적 내용이 없습니다. |
| 가이드라인 준수 | 플랫폼의 커뮤니티 가이드라인과 이용약관을 지키고 있습니다. |

<Warning>
위 항목 중 하나라도 허위로 체크한 경우, 수익화가 **즉시 차단**되고 이미 받은 피스가 **회수**될 수 있습니다.
</Warning>

---

## 수익화가 차단되는 경우

관리자는 다음에 해당하는 콘텐츠의 수익화를 차단할 수 있습니다.

| 사유 | 설명 |
|------|------|
| 2차 창작 | 웹소설, 만화, 게임 등 기존 작품을 기반으로 만든 콘텐츠 |
| 성인 콘텐츠 | 19세 미만이 볼 수 없는 성적/폭력적 콘텐츠 |
| 저작권 문제 | 권리가 없는 이미지, 텍스트, 음악 등을 사용한 경우 |
| 부적절한 내용 | 폭력, 혐오, 차별적 표현이 들어간 콘텐츠 |
| 가이드라인 위반 | 플랫폼 커뮤니티 가이드라인을 어긴 콘텐츠 |
| 관리자 판단 | 기타 운영팀의 판단에 따른 제한 |

수익화가 차단된 콘텐츠는 **무료로만 공개**할 수 있으며, 유료 판매를 통한 수익은 발생하지 않습니다.

## 피스 교환하기

### 하트로 바꾸기

1. **스튜디오 > 수익 관리** 페이지에서 현재 피스 잔액을 확인합니다.
2. **전환하기** 버튼을 누릅니다.
3. 바꿀 피스 수량을 입력합니다. (최소 100피스부터 가능)
4. 바뀐 하트는 **무료 하트**로 바로 지급됩니다.

<Info>
교환 비율은 **1피스 = 1하트**이며, 교환은 즉시 처리됩니다.
</Info>

### 수익 현황 확인하기

스튜디오의 수익 관리 페이지에서 다음 정보를 한눈에 확인할 수 있습니다.

- **현재 피스 잔액** - 교환 가능한 피스 수
- **총 누적 수익** - 지금까지 모은 전체 피스 수
- **총 교환 금액** - 하트로 바꾼 피스 수
- **월별 수익 추이** - 월 단위 수익 그래프
- **수익원별 비율** - 챕터/이미지/대화별 수익 비율
- **인기 콘텐츠 순위** - 수익이 높은 시나리오/캐릭터
- **거래 내역** - 전체 피스 입출금 기록

---

## 수익을 높이는 팁

1. **첫 챕터는 무료로** - 도입부를 무료로 제공해서 이용자의 관심을 끌고, 이후 챕터를 유료로 설정하세요.
2. **다양한 캐릭터 이미지** - 여러 표정과 상황에 맞는 이미지를 추가하고, 특별한 이미지는 유료로 설정하세요.
3. **꾸준한 업데이트** - 새로운 챕터와 캐릭터를 꾸준히 추가하면 재방문율이 높아집니다.
4. **태그를 잘 활용하세요** - 적절한 태그를 달면 검색에 더 잘 노출됩니다.
5. **반드시 공개 상태로** - 비공개 상태에서는 수익이 발생하지 않습니다. 완성 후 꼭 공개해주세요.

---

## 꼭 알아두세요

- **2차 창작은 수익화 불가** - 기존 작품 기반 콘텐츠는 무료 공개만 가능합니다.
- **실존 인물 기반은 제한** - 실존 인물을 다룬 콘텐츠는 수익화가 제한됩니다.
- **성인 콘텐츠는 현재 제한** - 19+ 콘텐츠는 현재 수익화가 제한됩니다.
- **부정 행위 절대 금지** - 자작자연(셀프 구매), 다중 계정을 이용한 부정 수익 등이 발견되면 계정 정지 및 수익 회수 조치가 이루어집니다.
- **수익 분배 실패 시 자동 재시도** - 일시적 오류로 수익 분배가 실패하면, 시스템이 자동으로 최대 5회까지 다시 시도합니다.
